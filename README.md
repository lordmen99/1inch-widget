## 1inch Embed Widget

### How to use

1. Create a `div` with `id` `1inchWidget` and link the JS file from the code below.

```js
<!DOCTYPE html>
<html lang="en">
  <head>
    <meta charset="UTF-8"/>
    <script src="https://raw.githubusercontent.com/nnnkit/1inch-widget/master/release/bundle.js"></script>
  </head>
  <body>
    <div id="1inchWidget"></div>
    <script >
      1inchWidget.init()
    </script>
  </body>
</html>
```
